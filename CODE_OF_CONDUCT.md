# Code of Conduct

## Our Pledge
We are committed to providing a welcoming, respectful, and harassment-free
environment for everyone, regardless of experience or background.

## Our Standards
Examples of positive behavior include:
- Being respectful and kind
- Giving constructive feedback
- Helping others learn

Unacceptable behavior includes:
- Harassment or discrimination
- Offensive language
- Personal attacks

## Enforcement
Project maintainers are responsible for enforcing this code.
Any unacceptable behavior may result in removal from the community.

## Scope
This Code of Conduct applies within all project spaces including GitHub and GitBook.

This Code of Conduct is adapted from the Contributor Covenant.
