## How to Contribute

You can contribute by:
- Fixing typos or improving explanations
- Adding examples or clarifications
- Proposing new documentation sections
- Improving structure and readability

## Steps
1. Fork the repository
2. Create a new branch (`docs/improvement-name`)
3. Make your changes
4. Commit with a clear message
5. Open a Pull Request

## Guidelines
- Keep content beginner-friendly and educational
- Use clear and simple language
- Avoid promotional or commercial content
